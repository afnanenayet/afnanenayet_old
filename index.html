    Welcome to Minigma

    Minigma is a small basic attempt at encryption. A seed is generated based on a user input which creates a cypher which can be used to encode/decode messages. The entire program is local, with no network access whatsoever.

    Installation Requirements

    How does it work?

    The program generates a seed based on a user input (the password). This password is converted to an integer which is used as the seed for the pseudo-random generator which manipulates an array to create a cipher. A "standard" cipher is used to convert the characters entered by the user into the corresponding characters in the generated cipher. This places the entire burden of security on the password rather than the system.
