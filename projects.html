My projects

This page contains and describes some of the interesting open source projects that are available for viewing and use on my Github profile.


Github profile


T3AI

A C++ application that allows two users to play tic-tac-toe, or for a user to play tic-tac-toe against an AI. The app can also detect when a game is over or if the game has been won.


View source on Github


REmap

A C++ application that tries to determine the location of cut sites from restriction fragment enzymes from the length of the fragments that they create.


View source on Github


Safe Lead

An OS X application written in Swift that analyzes certain statistics of a basketball game (time remaining, points that each team has and which team has possession), and determines whether the team that is in the lead has a safe lead. If the lead is safe, then it is unlikely that the team in the lead will lose the game. If the lead is not safe, there is a significant chance that the team in the lead could end up losing the game.


View source on Github


Corepair

A OS X application written in Swift that serves as a GUI front end for the file permissions verify/repair module that Apple removed from the Disk Repair application in OS X 10.11 El Capitan.


View source on Github


Minigma Library

The C++ implementation of the encryption scheme used in the Minigma Android app. This application creates a cipher based on a seed hashed from a user password. It uses the cipher to encode and decode a string.


View source on Github


Game Concept

A slightly modified version of some of the standard assets/examples included in the Unity Editor. This WebGL application represents a game concept: the car chase, from the perspective of the car that is chasing you. The objective of this game would be to get away from the car, while the camera was fixed from the chaser's perspective.
